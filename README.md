# iDine
## What is it?
A practice app made on SwiftUI following instructions on [hackingwithswift](https://www.hackingwithswift.com/quick-start/swiftui/whats-in-the-basic-template). 
With the app, users can browse information about a list of food, select what they want, and preceed to checkout.
